# cybersecurity_notes
This repository is actively updated as I learn new techniques, complete labs, and deepen my understanding of offensive and defensive security concepts. Everything here is practiced exclusively in authorized, legal lab environments.

All techniques, tools, and commands documented here are used exclusively within authorized lab environments such as Hack The Box, TryHackMe, and personal virtual machines. Unauthorized use of these techniques against systems you do not own or have explicit permission to test is illegal and unethical.
This repository is for educational purposes only.

# Key Tools Referenced
**Nmap** — Port scanning & service detection

**Gobuster / Feroxbuster** — Directory & subdomain brute-forcing

**Nikto** — Web vulnerability scanning

**Metasploit** — Exploitation framework

**Hydra** — Credential brute-forcing

**LinPEAS** — Linux privilege escalation enumeration

**Burp Suite** — Web traffic interception & manipulation

**SearchSploit** — Offline CVE/exploit search
